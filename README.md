# Paul Arthur Meteng — Portfolio

Personal portfolio website built with Next.js and Tailwind CSS.

**Live:** [arthur.meteng/portfolio](https://portfolio-blush-rho-65.vercel.app/)

## Tech Stack

- [Next.js 14](https://nextjs.org/) — App Router, static export
- [Tailwind CSS](https://tailwindcss.com/) — styling
- [TypeScript](https://www.typescriptlang.org/)

## Sections

- **Hero** — intro, CTA, social links
- **About** — bio, languages, background
- **Skills** — AI/LLM, backend, frontend, DevOps
- **Projects** — RAG system, thesis, conversational AI, web projects
- **Experience** — work history and education timeline
- **Contact** — email, LinkedIn, GitHub

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

## Deploy

Easiest option: [Vercel](https://vercel.com/) — connect the repo and it deploys automatically on every push.
